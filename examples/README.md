# Ejemplos

Esta carpeta contiene ejemplos de uso del paquete maquinay.

## Archivos

- `ejemplo_basico.py`: Ejemplos básicos de uso de las funciones principales
- `ola.py`: Ejemplo simple de "Hola mundo"

## Ejecutar los ejemplos

### Ejemplo básico

```bash
python examples/ejemplo_basico.py
```

### Hola mundo

```bash
python examples/ola.py
```
